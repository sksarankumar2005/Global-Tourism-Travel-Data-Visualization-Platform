# 🌍 Global Tourism & Travel Data Visualization Platform

An interactive **Power BI dashboard** that visualizes global tourism trends, holiday patterns, and travel statistics across 266 countries from 2016 to 2025.

---

## 📌 About the Project

This platform helps analyze global tourism data through two interactive dashboards — one for **stakeholders** (policymakers, tourism boards, travel agencies) and one for **travelers** (destination discovery, seasonal planning, safety metrics).

---

## 📁 Files in this Repository

| File | Description |
|------|-------------|
| `Dataset.xlsx` | Cleaned tourism dataset (2660 records, 29 features, 266 countries) |
| `Tourism and Travel _ Stakeholder Dashboard.pbix` | Power BI dashboard for stakeholders |
| `Tourism and Travel _ User Dashboard.pbix` | Power BI dashboard for travelers/users |

---

## 📊 Dashboards

### Dashboard 1 — Stakeholder Intelligence
> For tourism boards, policymakers, and travel agencies

- Global arrivals & receipts trend (2016–2025)
- Top countries by tourist arrivals (Filled Map)
- Tourism GDP impact and profit/loss analysis
- Key influencers driving tourism receipts (AI Visual)
- Decomposition tree for drill-down analysis
- Seasonal holiday patterns heatmap

### Dashboard 2 — Traveler Insights
> For individual travelers and destination researchers

- Safety, health & pollution index by country
- Best seasons to visit (holiday density)
- Hotel rates and average expenditure comparison
- Visa-free access percentage by country
- Air connectivity treemap
- Country scorecard — safety, cost, popularity

## 🛠️ Tools Used

- **Microsoft Power BI Desktop** — Dashboard development
- **Python (pandas)** — Data cleaning & preprocessing
- **Microsoft Excel** — Dataset storage

---

## 🚀 How to Open the Dashboards

1. Install [Power BI Desktop](https://powerbi.microsoft.com/desktop/) (free)
2. Download the `.pbix` files from this repo
3. Open in Power BI Desktop
4. Click **Refresh** and point to `Dataset.xlsx` if prompted

---

## 📈 Dataset Overview

- **266** countries & regions
- **10 years** of data (2016–2025)
- **29 features** including arrivals, receipts, expenditure, safety index, visa openness, holiday seasons, and more
